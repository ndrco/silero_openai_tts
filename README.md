# silero_openai_tts

## Где хранятся модели Silero

По умолчанию `torch.hub` (через который загружается Silero) сохраняет файлы в кэш пользователя:
- обычно `~/.cache/torch/hub` (Linux/macOS);
- или в каталог из переменной окружения `TORCH_HOME`, если она задана.

В этом проекте добавлен постоянный каталог `models/` в корне репозитория.
При старте приложения движок устанавливает `TORCH_HOME=models` и, если модели отсутствуют,
скачивает их автоматически в этот каталог.

Настройка каталога вынесена в `SILERO_MODELS_DIR` (по умолчанию `models`).
